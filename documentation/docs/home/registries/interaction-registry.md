# Interaction Registry

Use `@RegisterInteraction` and `@GenerateCodec`.

# How to register an interaction

## 1. Annotate your interaction class

Add `@RegisterInteraction("<your-interaction-id>")` and `@GenerateCodec` to your interaction class.

```kotlin
@RegisterInteraction("SimpleProjectileLaunchInteraction")
@GenerateCodec
open class SimpleProjectileLaunchInteraction : SimpleInstantInteraction() {
    //...
}
```

## 2. Add a companion object

The companion object must implement `CodecProvider<YourInteraction>` and provide the generated `CODEC`.

```kotlin
@RegisterInteraction("SimpleProjectileLaunchInteraction")
@GenerateCodec
open class SimpleProjectileLaunchInteraction : SimpleInstantInteraction() {
    companion object : CodecProvider<SimpleProjectileLaunchInteraction> {
        override val CODEC: BuilderCodec<SimpleProjectileLaunchInteraction> =
            CodecBuilderSimpleProjectileLaunchInteraction // (1)!
    }
}
```

1. This is generated by the compiler. The name follows the pattern `CodecBuilder<YourInteractionClassName>`. If it is not resolvable, build or run the project first.

## 3. Mark fields for serialization

Annotate each field you want serialized with `@CodecProperty`. Fields must be mutable (`var`).

```kotlin
@RegisterInteraction("SimpleProjectileLaunchInteraction")
@GenerateCodec
open class SimpleProjectileLaunchInteraction : SimpleInstantInteraction() {
    companion object : CodecProvider<SimpleProjectileLaunchInteraction> {
        override val CODEC: BuilderCodec<SimpleProjectileLaunchInteraction> =
            CodecBuilderSimpleProjectileLaunchInteraction
    }

    @CodecProperty(documentation = "The projectile to launch.")
    @CodecRequiredValidator
    var projectileId: String = ""
}
```

## 4. Disabling an interaction

Pass `enabled = false` to disable an interaction without removing it. The code generator will skip registration and log a warning at startup.

```kotlin
@RegisterInteraction("SimpleProjectileLaunchInteraction", enabled = false)
@GenerateCodec
open class SimpleProjectileLaunchInteraction : SimpleInstantInteraction() {
    //...
}
```

# Register interactions in your plugin

Call `InteractionRegistryGenerated.registerAll` in your plugin's `setup` method.

!!! warning
    If `InteractionRegistryGenerated` is not available, either the project has not been built yet or the `ksp{}` block in `build.gradle.kts` is misconfigured. Check for compiler errors when running the project.

```kotlin title="ExamplePlugin.kt" hl_lines="3-5" linenums="1"
class ExamplePlugin(init: JavaPluginInit) : JavaPlugin(init) {
    override fun setup() {
        InteractionRegistryGenerated.registerAll(
            registry = this.getCodecRegistry(Interaction.CODEC)
        )
    }
}
```
